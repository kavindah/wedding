== Description ==
Krystal is a WordPress theme built for most popular drag and drop page builder Elementor. Krystal comes with 8+ unique demos and with 1 click demo import feature. Krystal is SEO friendly, mobile responsive, translation ready theme and is compatible with most of the popular plugins. We also provide support to the free version. More information : https://www.spiraclethemes.com/krystal-free-wordpress-theme/ 


== Frequently Asked Questions ==

= Installation =
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Krystal in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' button to install and 'Activate' button to use.

= Where can I find the documentation of this theme?

1. Go to https://www.spiraclethemes.com/krystal-documentation/
